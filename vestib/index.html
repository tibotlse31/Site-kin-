VestibLab VPPB
Plateforme pédagogique structurée par modules : diagnostic, traitements, cas cliniques, checklists, pièges et visualiseur d’appoint.

Le cours passe avant l’animation

Cette version sépare les outils en fenêtres claires. Le diagnostic et les contenus de cours sont centraux. Le visualiseur n’est plus la page principale : il devient un support pour comprendre les tests diagnostiques.

Arbre décisionnel Manœuvres pas à pas Cas corrigés Checklists Red flags

Utilisation conseillée

  1. Commencer par Diagnostic pour maîtriser la lecture clinique.
  2. Poursuivre par Traitements pour apprendre les séquences.
  3. S’entraîner avec les Cas cliniques.
  4. Réviser avec les Checklists et les Pièges.
  5. Utiliser le Visualiseur seulement comme appui visuel.
